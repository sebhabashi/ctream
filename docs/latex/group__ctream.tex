\doxysubsection{Detailed Description}
This group contains templated class \mbox{\hyperlink{classctream_1_1internal_1_1Ctream}{Ctream}} which is the entrypoint of this library.

\mbox{\hyperlink{classctream_1_1internal_1_1Ctream}{Ctream}} provides a clear syntax and parallelized implementation for inline data manipulation. \mbox{\hyperlink{classctream_1_1internal_1_1Ctream}{Ctream}} objects can be created using diverse STL containers as source, then modified through data mapping extraction and filtering, and then exported using Collector\} objects into usable types.
